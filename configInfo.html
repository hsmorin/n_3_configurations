Computation Parameters

Line field of slope 1:

The line field upon which the relative position of the eigenvalues with regards to braiding is determined is made up of lines of slope 1. More concretely, in the program the variable is set as:
projectionDirection = [-1, 1]
 which represents $-x+y$, or in this case we are talking about numbers in the complex plane so for each $z \in \mathbb{C}$ it represents $-\mathrm{Re}(z)+\mathrm{Im}(z)$
So the program assigns a single real number, $\alpha$, to each complex eigenvalue such that the eigenvalue's position in the complex plane corresponds to a point in $\mathbb{R}^2$ lying on the line
$y-x=\alpha$
and so by comparing these $\alpha$ you get an ordering of the eigenvalues
For the braid diagram, the reversed order setting actually recomputes the braid with projectionDirection = [1, -1], effectively causing the $\alpha$ value to be multiplied by $-1$ thus reversing the ordering of all eigenvalues

Over/Under is determined by the eigenvalue with the smaller real component:

When two eigenvalues cross, the eigenvalue with the smaller real component is considered as crossing over by default. Toggling swapped over/under on the braid diagram changes it so that the eigenvalue with the greater real component is over.

The program starts at $t=0.0000000000000001$ and ends at $t=1$:

All computations do not technically start at $t=0$. They have to start slightly after 0 so that the eigenvalues are slightly seperated instead of being directly on top of each other, which is required for them to be colored consistently. Otherwise the colors are assigned at random every time because each eigenvalue in a cluster starts exactly the same.

$q_1$ and $q_2$ follow a quarter circle path:

At a given $t$,
$q_1 = (\sin(\frac{\pi t}{2})$ + perturbation($t, \varepsilon))^n$
$q_2 = (\cos(\frac{\pi t}{2}))^m$
Where $n$ and $m$ are the weights such that the quantum spectrum polynomial is homogeneous by $\lambda \cdot (q_1,q_2,t) = (\lambda^n q_1, \lambda^m q_2, \lambda t)$

$\varepsilon=0.15$

perturbation$(t, \varepsilon) = \sqrt{\sin(\pi t)} \cdot \varepsilon \cdot i$
Thus the perturbation starts at 0, so the clustering is clear, then steeply increases so that the roots become perturbed almost immediately, then sharply drops to 0 again at $t=1$.
The negative perturbation setting sets $\varepsilon = -0.15$
